Custom Radio Buttons

To create a custom radio button, wrap a container element, like <div>,
1. with a class of .custom-control and .custom-radio around the radio button.
2. Then add the .custom-control-input to the radio button.

Tip: If you use labels for accompanying text, add the .custom-control-label class to it.
