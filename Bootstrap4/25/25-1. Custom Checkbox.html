Custom Checkbox

To create a custom checkbox,
- Wrap a container element, like <div>, with a class of .custom-control class and .custom-checkbox class around the checkbox.
- Then add the .custom-control-input class to the checkbox.

Tip: If you use labels for accompanying text,
- Add the .custom-control-label class to it.
