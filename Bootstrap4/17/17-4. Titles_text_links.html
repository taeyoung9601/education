Card titles, text, and links

Use .card-title class to add ( card titles ) to any ( heading ) element.
The .card-text class is used to remove bottom margins for a ( p ) element if it is the ( last child or the only one ) in .card-body.
The .card-link class adds a ( blue color ) to any ( link ), and a ( hover effect ).
