Inline Flex

To create an inline flexbox container, add the .d-inline-flex class.
Use any of the .d-*-inline-flex classes to control WHEN the element should be a flex element on a specific screen width.

Resize the browser window to see the effect:

.d-inline-flex
.d-sm-inline-flex
.d-md-inline-flex
.d-lg-inline-flex
.d-xl-inline-flex
