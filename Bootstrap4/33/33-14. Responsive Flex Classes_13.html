Order

Change the visual order of a specific flex item(s) with the .order classes.
Valid classes are from 0 to 12.

Use .order-sm|md|lg|xl-0-12 classes to achieve this effect on different screen sizes.
The classes can also be combined to make an even more flexible layout.
