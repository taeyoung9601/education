Contextual Classes

Contextual classes can be used to color the table, table rows or table cells.
The classes that can be used are:

  • .table-primary
  • .table-success
  • .table-info
  • .table-warning
  • .table-danger
  • .table-active
  • .table-secondary
  • .table-light
  • .table-dark
