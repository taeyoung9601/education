Input Groups

The .input-group class is a container to enhance an input by adding an icon, text or a button
in front .input-group-prepend class or behind .input-group-append the input field as a help text.

Use the .input-group-text class to style the specified help text.

*Tip*: We use the .mb-3 utility class to ensure that the input group gets a proper margin bottom.
