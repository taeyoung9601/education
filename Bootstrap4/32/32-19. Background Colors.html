Contextual Backgrounds

Use the contextual background classes to provide "meaning through colors".

Note that you can also add a .text-* class if you want a different text color:

This text is important. ( .text-white )

This text indicates success. ( .text-white )

This text represents some information. ( .text-white )

This text represents a warning. ( .text-white )

This text represents danger. ( .text-white )

Secondary background color. ( .text-white )

Dark grey background color. ( .text-white )

Light grey background color. ( .text-dark )
