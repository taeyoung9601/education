pt, my 공부

pt는 padding top으로 위쪽 패딩을 의미 ( 현재는 pt-3 ).

my는margin-y의 약자로 수직여백, 위쪽과 아래쪽 margin 의미 현재는 my-5

bg 공부

-bg 는 background의 약자

-danger는 붉은 계열 , warnning 노란 계열, primary 푸른계열 등등 다양한 색 표현가능 뒤에 -subtle 붙으면 색이 연해짐.

text 공부

text-bg-warning을 입력했는데 검은색으로 글씨체가 나오는이유는 배경색과 대비를 위해 자동으로 색 변경

반응형 컨테이너

컨테이너의 최대 너비를 화면 크기에 맞게 자동으로 조절하는 역할

container-sm은 small의 약자로 면 너비가 576px 이상일 때 최대 너비를 적용
container-md은 medium의 약자로 화면 너비가 768px 이상일 때 최대 너비를 적용
.container-lg는 large의 약자로 .container-lg: 화면 너비가 992px 이상일 때 최대 너비를 적용
.container-xl는 extra large의 약자로 화면 너비가 1200px 이상일 때 최대 너비를 적용
