Bootstrap 4 Alerts

Alerts are created with the .alert class, followed by a contextual color classes:

.alert-success : This alert box could indicate a successful or positive action.
.alert-info : This alert box could indicate a neutral informative change or action.
.alert-warning : This alert box could indicate a warning that might need attention.
.alert-danger : This alert box could indicate a dangerous or potentially negative action.
.alert-primary : Indicates an important action.
.alert-secondary : Indicates a slightly less important action.
.alert-dark : Dark grey alert.
.alert-light : Light grey alert.

알림 링크

알림 박스 안에 있는 링크에 .alert-link 클래스를 추가하여 "색상에 맞는 링크"를 만들 수 있습니다.

.alert-info : You should read this message.
.alert-warning : You should read this message.
.alert-danger : You should read this message.
.alert-primary : You should read this message.
.alert-secondary : You should read this message.
.alert-dark : You should read this message.
.alert-light : You should read this message.

Alerts

클래스 class="close"와 data-dismiss="alert"가 적용된 버튼은 알림 박스를 닫는데 사용됩니다.

.alert-dismissible 클래스는 닫기 버튼에 추가적인 패딩을 제공합니다.

.alert-success : 이 알림 박스는 성공적이거나 긍정적인 행동을 나타낼 수 있습니다.
.alert-info : 이 알림 박스는 중립적인 정보성 변화나 행동을 나타낼 수 있습니다.
.alert-warning : 이 알림 박스는 주의가 필요한 경고를 나타낼 수 있습니다.
.alert-danger : 이 알림 박스는 위험하거나 부정적인 행동을 나타낼 수 있습니다.
.alert-primary : 중요한 행동을 나타냅니다.
.alert-secondary : 조금 덜 중요한 행동을 나타냅니다.
.alert-dark : 어두운 회색 알림입니다
.alert-light : 밝은 회색 알림입니다.

Animated Alerts
