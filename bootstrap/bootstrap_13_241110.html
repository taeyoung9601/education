Colored Progress Bars

Use any of the contextual color classes to change the color of the progress bar:

.progress-bar

.bg-success

.bg-info

.bg-warning

.bg-danger

.bg-white

.bg-grey

.bg-light

.bg-dark

Striped Progress Bars

The .progress-bar-striped class adds stripes to the progress bars:

.progress-bar-striped

.progress-bar-striped, .bg-success

.progress-bar-striped, .bg-info

.progress-bar-striped, .bg-warning

.progress-bar-striped, .bg-danger

Animated Progress Bar

Add the .progress-bar-animated class to animate the progress bar:

.progress-bar-striped, .progress-bar-animated
