Contextual Colors

Use the contextual classes to provide "meaning through colors":

.text-muted - 이텍스트는 "약간 흐릿하거나 흐려진" 또는 "덜 강조된" 표시입니다..

.text-primary- 이텍스트는 중요하다는 표시입니다..

.text-success- 이텍스트는성공적인 상태를 나타냅니다..

.text-info - 이텍스트는 정보를 나타냅니다..

.text-warning - 이텍스트는 경고를 나타냅니다..

.text-danger - 이텍스트는 위험을 나타냅니다..

.text-secondary - 이텍스트는 보조하는 역할입니다..

.text-dark - 이텍스트는 어두운 회색입니다..

.text-body - 이텍스트는 본문내용에 씁니다. color (often black).

.text-light - 이텍스트는 연한 회색입니다. 밝은 계열 배경에서

.text-white - 이텍스트는 흰색입니다. 밝은 계열 배경에서

Contextual Link Colors #2

링크위에 마우스올려보세요.

.text-muted - Muted link.
.text-primary - Primary link.
.text-success - Success link.
.text-info - Info link.
.text-warning - Warning link.
.text-danger - Danger link.
.text-secondary - Secondary link.
.text-dark - Dark grey link.
.text-body - Body/black link.
.text-light - Light grey link.

불투명도

Add 50% opacity for black or white text with the .text-black-50 or .text-white-50 classes:

.text-black-50 - Black text with 50% opacity on white background

.text-white-50, .bg-dark - White text with 50% opacity on black background

Contextual Backgrounds

문맥에 맞는 배경 클래스(contextual background classes)를 사용하여 '색상으로 의미를 전달하세요

다른 텍스트 색상을 원하면 .text- 클래스를 추가할 수도 있습니다

.bg-primary .text-white : This text is important.

.bg-success .text-white : This text indicates success.

.bg-info .text-white : This text represents some information.

.bg-warning .text-white : This text represents a warning.

.bg-danger .text-white : This text represents danger.

.bg-secondary .text-white : Secondary background color.

.bg-dark .text-white : Dark grey background color.

.bg-light .text-dark : Light grey background color.
